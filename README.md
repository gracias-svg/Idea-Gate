Idea Gate is not a prompt tool.

* 📊 Product Operating System

It simulates:

* Product Manager
* Designer
* Engineer

* GTM team

* Skip discovery

## ✅ Solution

Idea Gate enforces a **real product lifecycle**:

```
0 → Idea Intake
1 → Discovery
2 → Problem Definition
3 → Solution Design
4 → MVP Hypothesis
5 → Validation
6 → Prioritization
7 → PRD
8 → UX Design
9 → Usability
10 → Architecture
11 → Backlog
12 → Implementation Plan
13 → QA & Readiness
14 → Prototype Prompt
```

## 🧠 System Architecture

```
User → CLI
     → Coordinator (PM Brain)
     → Lifecycle Engine
     → Multi-Agent System
     → Journey Engine (Memory)
     → Artifacts
     → Prototype Prompt
```
